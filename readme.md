# ts-data-formatting-utils

This is a library for formatting data using only ts

## Table

### Formatted Table

yes.
example Input: `[{ name: "asdf", value: 1 }, { name: "something else indeed", value: 100 }]`

```
┌───────────────────────┬───────┐
│         name          │ value │
├───────────────────────┼───────┤
│ asdf                  │ 1     │
│ something else indeed │ 100   │
└───────────────────────┴───────┘
```
